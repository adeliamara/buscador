Uma classe em Python é uma estrutura de programação que permite definir um conjunto de atributos e métodos que descrevem um objeto. Uma classe é como um modelo ou uma planta para um objeto, e pode ser usada para criar instâncias desse objeto.

Para definir uma classe em Python, podemos usar a palavra-chave class seguida do nome da classe:


class MinhaClasse:
    # código da classe aqui

Dentro da classe, podemos definir atributos e métodos. Os atributos são variáveis que contêm dados sobre o objeto, enquanto os métodos são funções que realizam operações com esses dados.

Por exemplo, podemos definir uma classe Pessoa com os atributos nome e idade e o método dizer_nome:


class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade

    def dizer_nome(self):
        print("Meu nome é", self.nome)

Nesse exemplo, o método __init__ é um método especial que é executado quando uma instância da classe é criada. Ele inicializa os atributos da classe com os valores passados para ele. O argumento self refere-se à instância atual da classe.

Podemos criar uma instância da classe Pessoa usando o seguinte código:


pessoa1 = Pessoa("João", 30)

Agora podemos acessar os atributos e métodos da instância pessoa1 usando a notação ponto:


print(pessoa1.nome)  # saída: "João"
print(pessoa1.idade)  # saída: 30
pessoa1.dizer_nome()  # saída: "Meu nome é João"

Dessa forma, podemos usar classes em Python para criar objetos com atributos e métodos que nos permitem trabalhar com esses objetos de maneira mais eficiente e organizada.
